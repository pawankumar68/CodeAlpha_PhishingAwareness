# Phishing Awareness Training

A presentation built for the **CodeAlpha Cyber Security Internship - Task 2**, designed to educate people on recognizing and avoiding phishing attacks.

## 📌 Description

This presentation covers the fundamentals of phishing attacks, including:

- What phishing is and what attackers aim to steal
- Different types of phishing (email phishing, spear phishing, smishing/vishing, clone websites)
- Social engineering tactics used to manipulate victims (urgency, authority, fear, curiosity)
- How to spot a phishing email and a fake website, with real-world examples
- Best practices to stay safe online
- An interactive quiz to reinforce learning

## 🛠 Tools Used

- Google Forms (for the interactive quiz)

## 🎯 Key Takeaways Covered

- Phishing relies on deception and urgency — always pause before acting
- Verify sender addresses, links, and requests independently
- Enable Multi-Factor Authentication (MFA) wherever possible
- When in doubt, don't click — report suspicious messages to IT/security teams

## 🎓 What I Learned

- How phishing attacks are designed to exploit human psychology rather than just technical vulnerabilities
- Practical methods to identify fake emails and fraudulent websites
- The importance of awareness training in preventing real-world cyber attacks
- How to build engaging, educational security content
